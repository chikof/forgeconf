# Security Vulnerabilities

If you discover a security issue in Forgeconf, please reach out privately via the Discord server at <https://discord.gg/sejc7TnX6N> (ping @chikof or a moderator). We aim to reply within two business days and will coordinate a fix as quickly as possible.

The repository uses Dependabot to keep transitive dependencies up to date. Nevertheless, responsible disclosure helps us protect downstream users.
